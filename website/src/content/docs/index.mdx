<h2 class="title">易于使用</h2>

SortAO 提供了开箱即用的四个高级包装器。

- [**Sorted-Map**](./associative/sorted-map/)：有序字典，键唯一。
- [**Sorted-Set**](./associative/sorted-set/)：有序集合，值唯一。
- [**Sorted-Multimap**](./associative/sorted-multimap/)：有序多重字典，允许重复键。
- [**Sorted-Multiset**](./associative/sorted-multiset/)：有序多重集合，允许重复值。

</section>
